# dashboard-navigation-and-user-interface-811-821

This workspace contains the PostgreSQL database for the social media dashboard.

Quick start:
- Run: `bash social_media_database/startup.sh`
- Connection: see `social_media_database/db_connection.txt`
- Schema and seed: auto-run by startup.sh; can be re-run via `bash social_media_database/sql_init.sh`
- Env template: `social_media_database/.env.example`
